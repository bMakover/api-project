Toy API Documentation

Toy API Endpoints

Endpoint Method Description Authorization Parameters / Query
/toys GET Fetches a paginated list of toys. perPage (default: 10), page (default: 1)
/toys/search GET Searches for toys based on name or info. s (search term)
/toys/category/:category GET Retrieves toys of a specific category. category
/toys/single/:id GET Fetches a single toy by its ID. id
/toys/prices GET Retrieves toys within a specified price range. min (minimum price), max (maximum price), perPage (default: 10), page (default: 1)
/toys POST Creates a new toy. Required Toy information
/toys/:editId PUT Updates an existing toy by ID. Required Updated toy information
/toys/:delId DELETE Deletes a toy by ID. Required delId (toy ID)

User API Endpoints

Endpoint Method Description Authorization Parameters / Query
/users GET Displays a message confirming user operations.
/users/myInfo GET Retrieves the information of the authenticated user.
/users/usersList GET Retrieves a list of users. Required for admin users
/users POST Registers a new user. User information
/users/login POST Authenticates a user and generates a token for authorization. User login credentials
