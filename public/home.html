Toys api documentation

Routes

All HTTP methods are supported. You can use http or https for your requests.

  • Get 10 toys by page number:/toys?page=number (GET)
  • Search toys by name or info and page number:/toys?s=key-word&page=number (GET)
  • Get all toys by category and page number:/toys/category/:category-name?page=number (GET)
  • Get single toy by id:/toys/single/:toyId (GET)

    • Add toy - (only for register user - must send user's auth-token through headers):/toys (POST)
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)
  • user_id (required)
  • date_created

    • Edit toy - (only for register user - must send user's auth-token through headers):/toys/:_id (toy) (PATCH)
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)
  • user_id (required)
  • date_created

    • Delete toy - (only for register user - must send user's auth-token through headers):/toys/:_id (toy) (DELETE)

    Sign Up:/users/ (POST)
  • name (required)
  • email (required)
  • password (required)
  • role

    • Sign Up:/users/login (POST)
  • email (required)
  • password (required)
